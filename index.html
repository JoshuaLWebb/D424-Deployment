Download the WGU mobile app

Read the instruction below to install the latest version of WGU mobile app on your Android device.

How to install:
  1. Scan the QR code above if you're on a desktop
  2. Tap the button below to download the APK file.
  3. If prompted with a warning, select "Download anyway"
  4. Once the download completes, click "Open". This will take you to your downloads.
  5. Next, tap the "MobileApp.apk" file.
  6. If your phone is set to not install apps from unknown sources, a popup will appear. Tap "Settings" > Toggle "Allow permission"
  7. Tap install to install the Mobile App
  8. Once installed, log in with username: admin and password: admin to view demo data.
